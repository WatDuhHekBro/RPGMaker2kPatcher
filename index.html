Instructions

Drag and drop files here to make the magic happen.

The accepted file types are .lmu, .ldb, .json, and .patch.json.

Upload an .lmu file here and the parsed data will be downloaded along with a fill-in-the-blank patch and a common characters file if you're importing

Upload an .ldb file and it'll do the same as an .lmu file except that a common characters file will also be downloaded.

Uploading a .json or .patch.json file won't do anything immediately, but will be stored. When you click the button below, it'll deal with these files.

.json files will generate an .lmu file. If a patch exists, then the map file will be patched.

.patch.json files will display dialogue info in the console if no corresponding map exists and patch a map file if it exists. It'll also show copy-paste-ready text for Google Translate.

Info about which maps got patched and which ones didn't will be in the console. The text will also be available in the console.
